# Hemanth Kumar K - Premium Portfolio

A highly premium, interactive, and beautifully designed personal portfolio website showcasing my expertise in Embedded Systems, IoT, and Full-Stack Digital Experiences.

Built with modern web technologies and designed with a strict focus on sleek monochrome aesthetics, fluid animations, and a dynamic tracking glow effect.

## 🌟 Live Website
**[https://hemanthnanu-tech.github.io/Hemanth-Kumar-K-protfolio/](https://hemanthnanu-tech.github.io/Hemanth-Kumar-K-protfolio/)**

## 🛠 Technologies & Architecture
- **React 18**: Component-based UI architecture
- **Vite**: Blazing fast build tool and development server
- **Tailwind CSS**: Utility-first styling with custom glassmorphism extensions
- **Framer Motion**: Smooth, physical animations and scroll reveals
- **Lucide & Devicons**: Clean, professional iconography

## ✨ Key Features
- **Strict Monochrome Theme:** A beautiful high-contrast black-and-white aesthetic that perfectly adapts to Light and Dark modes.
- **Dynamic 3D Infinite Marquee:** A futuristic, zero-interaction scrolling technical arsenal displaying colorful software and hardware skills.
- **Mouse Tracking Glow:** A soft, custom ambient light that perfectly tracks the user's cursor across the entire page, completely rewritten for zero-lag 60fps performance on both desktop and mobile devices.
- **Direct WhatsApp Integration:** A seamlessly integrated contact form that pre-formats user inquiries and launches directly into a WhatsApp conversation.
- **Premium Global Typography:** Fully integrated with the `Outfit` font for a high-end editorial feel, paired with a glitch-effect typographic preloader.
- **Categorized Projects:** Distinct sections separating full-stack Software Apps from Academic/Hardware prototypes.
- **Interactive Education Timeline:** Educational history linked directly with official certification and verification documents via Google Drive.

## 🚀 Local Development
To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/hemanthnanu-tech/Hemanth-Kumar-K-protfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd Hemanth-Kumar-K-protfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Deployment
This project is fully automated and configured to deploy to **GitHub Pages** using GitHub Actions whenever a change is pushed to the `main` branch.
